# Interactive Personal Data Collector

A simple Python CLI program that collects and confirms user personal
information through an interactive terminal interface.

## 📌 Project Description

This project is a **command-line based personal data collector** written
in Python. It asks the user to input personal details and then confirms
the information before saving the final response.

If the information is incorrect, the program allows the user to re-enter
the details until they confirm accuracy.

The program runs in a loop until the user confirms the entered
information.

## 🚀 Features

-   Interactive terminal interface
-   Collects multiple user details:
    -   Name
    -   Age
    -   Email
    -   Phone number
    -   Address
-   Confirmation step before final submission
-   Allows re-entry if information is incorrect
-   Simple and beginner-friendly Python project

## 🛠️ Technologies Used

-   Python 3
-   Command Line Interface (CLI)

## ▶️ How to Run

1.  Install Python 3 on your system.
2.  Download the project file.
3.  Run the program:

``` bash
python proj1.py
```

4.  Enter your details when prompted.
5.  Confirm the information.
